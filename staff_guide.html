koya 総合スコアとは？

スコアの定義

「koya 総合スコア」は、米国マクロ経済・在沖米軍情勢・TRICARE 医療需要など 21 の指標を統合した 0〜100 点の経営判断指標です。
スコアが高いほど「今この瞬間に積極的にサービスを展開すべき」市場環境であることを示します。

FTO サービス価格シートの見方（21 業種）

価格シートの列構成

Googleスプレッドシートの「FTOサービス価格」シートには下記の列があります。

列名 意味 活用例
業種キー / 業種名 サービスの識別子と名称 「ft_2d」= 2D エコー、「cafe」= カフェ 等
カテゴリ FTO / 飲食 / 生活 / 美容 / 小売 / 体験 / 教育 サービス分類・販促の軸として使用
米US低 / 米US高（$） アメリカ国内での同等サービスの相場（最低〜最高） 「アメリカより断然お得！」な比較訴求の根拠
CPI 低 / CPI 高 米国物価(CPI)で調整後の実質相場感 インフレ考慮後の最新相場として参照
FTC 参考（¥） 日本国内の標準的な市場価格（円） 日本市場との比較・競合分析に使用
FX（円/ドル） データ取得時の為替レート（現在: 161.8円） 円換算計算の基準値
円低 / 円高（¥） 米国相場を現在の為替で円換算した範囲 「同じサービスをアメリカで受けると ¥X 〜 ¥Y」の参考値
推奨低 / 推奨高（¥） koya が設定する推奨販売価格帯（円） 実際の販売価格設定・見積もりの基準
月額/人 月間利用見込み人数 収益シミュレーションの計算に使用
市場M（百万円） 沖縄在沖米軍市場の推定市場規模 優先投資先サービスの判断材料
価格提案のポイント： 「推奨低〜推奨高」の価格帯は円安 + 米国インフレ + koya の価値訴求を総合的に加味した価格です。 軍属のお客様に価格について聞かれた場合、「アメリカだと $X〜Y（約 ¥___）するサービスを、 日本でこのクオリティで提供できる」と説明すると納得感が高まります。
Section 05
Googleスプレッドシートへのアクセス方法
1

スプレッドシートを開く

下記リンクをクリックすると、koya 市場データベース（Googleスプレッドシート）が開きます。Googleアカウントへのログインが必要です。

2

シートタブを切り替える

画面下部のタブで「ダッシュボード」（21 指標一覧）と「FTOサービス価格」（21 業種価格表）を切り替えられます。

3

ダッシュボードを確認する

「指標」「値」「スコア」「アラート」の列をチェックします。green = 良い / yellow = 注意 / red = 警戒で色を読み取ります。

4

価格表を参照する

「FTOサービス価格」シートで担当サービスの「推奨低〜推奨高（¥）」列を確認します。この価格はデータ更新のたびに最新化されます。

5

ダッシュボードサイトも活用する

よりビジュアルで詳細なデータはライブダッシュボードサイトで確認できます。タブ別に「市場概況」「価格心理」「住宅分析」「GEO情勢」などが見られます。

Section 06
スタッフ向け FAQ

Q. データはいつ更新されますか？

Googleスプレッドシートのデータは、GAS（Google Apps Script）スクリプトを実行するたびに最新化されます。 通常は週1回〜月数回のペースで更新が行われます。シートの「更新日」列で最終更新日を確認できます。

Q. ドル円が下がったら価格はどうなりますか？

為替が円高（例: 130円/ドル）になると、軍属のドル収入の円換算が下がるため購買力が低下します。 推奨価格も次回データ更新時に見直されます。スプレッドシートの「FX」列の数値が下がってきたら 注意のサインです。消費者信頼感や VIX も合わせて確認してください。

Q. 基地再編リスク「42点（red）」は何が心配？

42点は「基地縮小・移転のリスクが中程度に存在する」ことを示します。現時点で緊急ではありませんが、 在沖米軍の人員削減が発表された場合は顧客層に影響が出る可能性があります。 半年に1度は GEO タブの数値を確認し、変化があれば経営層に報告をお願いします。

Q. FTO カテゴリと他カテゴリの違いは？

「FTO」カテゴリ（2Dエコー・4Dエコー・シッター・産後ケア）は Family Tree Oki が 直接提供するコアサービスです。「飲食」「美容」「体験」などは連携パートナー・ エコービジネス候補として市場規模を把握するための参照データです。 koya スコアに最も直結するのは FTO カテゴリの指標です。

Q. GitHub Pages の CSV データとは何ですか？

最新の市場データは下記 URL でCSV形式でも公開されています。Googleスプレッドシートに IMPORTDATA 関数でリンクして自動更新の活用も可能です。
